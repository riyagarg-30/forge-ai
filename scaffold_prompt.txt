# Files to Create

Create the following files inside:

backend/src/scaffold/

### agentTemplate.js

Purpose:

Provide a reusable template that every future AI agent can start from.

Requirements:

- Use parseContext(context)
- Follow the same context structure as existing agents
- Include placeholders for:
  - summary
  - analysis
  - findings
  - recommendations
  - risks
  - readinessSignals

This file should serve only as a template.

---

### contextHelpers.js

Create reusable helper functions for extracting commonly used business details.

Examples:

- industry
- targetRegion
- businessModel
- revenueModel
- startupStage
- targetAudience

Provide sensible default values where appropriate.

Keep the helpers generic.

---

### responseTemplate.js

Create reusable helper functions that generate a consistent response structure for every future agent.

The goal is to ensure every agent returns data in a predictable format while allowing agent-specific fields to be added later.

---

### scoring.js

Create generic scoring utilities.

Examples:

- normalizeScore()
- readinessLevel()
- classifyRisk()

These utilities must NOT contain logic specific to any individual agent.

Keep them reusable across all current and future agents.
